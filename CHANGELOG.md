# Changelog

本项目的所有主要更改都将记录在此文件中。

## [v2.1.2] - 2026-06-07

### Added
- **全新 Now Playing 皮肤：Apple Style**：新增 Apple 风格 Now Playing 皮肤，采用 AMLL Mesh Gradient 动态渐变背景，极其接近 Apple 风格，视觉效果更沉浸。
- **模糊搜索**：基于 FTS5 全文索引的本地搜索，支持歌名、歌手、专辑名模糊匹配。
- **基于歌词的搜索**：搜索结果高亮显示曲库内歌词片段匹配，方便快速定位想听的歌。
- **外部播放源自定义优先级**：外部播放源支持自定义优先级排序，可根据个人习惯调整播放源的使用顺序。

### Changed
- **OKLCH 色彩系统全面升级**：主题色提取和渲染从 HSL 迁移到 OKLCH 色彩空间。LED 频谱可视化、歌词高亮色、封面模糊色、全屏氛围色等所有视觉元素都经过重新调校，色彩更自然、层次更丰富。
- **歌词系统增强**：LDDC 歌词元数据清理改进，翻译匹配更准确；导入歌曲支持提取歌曲文件自带 LRC 歌词
- **更新检查改进**：支持直接下载安装包，无需跳转浏览器。

### Fixed
- 主页卡顿性能问题
- 艺术背景转场卡顿问题
- 外观设置页面布局优化


## [v2.0.0] - 2026-05-11

### Added
- **全新 Home 页面（beta）**：新增 Home 页面，可快速查看专辑、艺人、播放列表、每周聆听和作品排行，也可进入全部专辑与全部艺人页面继续浏览。
- **封面与元数据网络补全**：新增网络数据源，导入或整理音乐时可更易找到合适封面，并可引入专辑信息和艺人背景信息。
- **AMLL 低分辨率渲染**：新增低分辨率渲染模式，大幅提高歌词流畅度。
- **AMLL 减弱动画选项（beta）**：新增可选减弱动画选项，高亮以离散方式亮起，更加安静。
- **外部播放支持（beta）**：支持在其他受支持的音乐 App 里播放，并在 kmgccc_player 中控制与欣赏。
- **全屏播放优化**：支持在小窗口中全屏（点击正在播放的歌曲封面即可进入），并支持在全屏下编辑显示效果和皮肤。
- **Dock 栏播放进度**：在设置中开启后，可在 macOS Dock 栏 App 图标上查看播放进度，右键图标可在菜单中快捷进行播放控制。
- **播放记忆**：重新打开 App 后，可保留上次的播放状态。

### Changed
- **颜色与主题系统优化**：封面取色和主题生成更自然，播放器会更细致地判断作品色彩，让界面、歌词、背景和 LED 的颜色更统一。
- **播放界面效果优化**：LED 模拟组件引入 OKLCH 颜色体系，更符合人眼观感；新增呼吸灯模拟老式播放器效果；CD 模式效果更自然。
- **资料库与数据管理升级**：资料库支持自定义位置；扫描、加载、补全、批量删除、多选和多艺人解析等流程更加完善。
- **智能随机播放与聆听记录**：继续优化智能随机播放和聆听习惯记录。
- **界面与设置统一**：工具栏、设置页、播放列表、滚动效果、卡片材质和 Sidebar 细节整理，整体界面更统一，更接近 macOS 体验。

### Fixed
- 项目迁移到 Swift 6，重构播放、导入、歌词、工具栏和全屏相关架构，修复大量稳定性问题。

## [v1.3.1] - 2026-04-14

### Added
- **AMLL DB 歌词查找**：接入 AMLL DB 作为歌词来源之一，扩大高质量歌词的检索覆盖范围。
- **播放列表增强**：播放列表详情页加入 Header，支持根据歌曲主色生成预置图片，也支持手动选择自定义封面。
- **导入后后台补全**：导入歌曲时可选在后台补全封面、歌词等信息，可以在“设置-数据”中控制。
- 资料库存储扩展：音乐资料库新增专辑、歌手维度的持久化存储。
- **随机播放偏好记录**：新增随机播放偏好学习能力，可根据历史聆听习惯动态调整随机队列。
- 动态排序能力：播放列表排序支持结合聆听偏好进行动态排序。
- **旋转封面 CD 模式**：新增高速旋转的 CD 风格封面模式，实现动态模糊，模拟 CD 播放器的视觉表现。

### Changed
- 歌词检索流程优化：歌词查找流程进一步简化，点击搜索结果可直接应用，无需再手动点击“应用”。
- 歌词转换链路重写：原有歌词转换脚本改为 Swift 原生实现。
- 封面查找交互改进：封面搜索结果支持手动挑选候选图，方便为歌曲选择更合适的封面; 部分封面可能需要特定网络环境才能加载。
- 全屏底部控制栏调整：重新整理底部控制栏布局，并支持自动隐藏。
- 全屏玻璃材质选项：全屏播放器新增玻璃材质选择开关，可在设置中选择。
- 菜单栏体验优化：调整菜单栏结构与交互细节。

### Improved
- 降低磁带皮肤与 `artbk` 背景相关内存占用。

### Fixed
- 修复音乐资料库反复写入导致歌曲封面反复压缩、画质劣化的问题。

- 频谱可视化优化：响应更灵敏，动画更顺滑，经典封面皮肤支持独立频谱显示。
- 音频设备变更恢复：连接耳机或切换音频设备时自动恢复播放，体验更连续。

### Improved
- 封面缓存与艺术背景渲染优化，内存占用更低，整体运行更流畅稳定。

## [v1.0.3] - 2026-02-14

### Notes 🌟🌟
- **更名带来的库路径变化需要手动迁移旧数据**：如你之前使用过该 app，请将旧库内容 改名为 `~/Music/kmgccc_player Library`,或者将旧库 `~/Music/TrueMusic Library` 内文件夹移动到新的 Library 文件夹下，否则 App 可能无法找到既有曲库。

### Added
- 🌸 新增艺术背景系统，封面风格化取色引擎，提供更美观的播放界面（还在开发中，目前效果/性能可能不稳定）。
- 新增播放列表歌曲多选操作，以及批量编辑元数据、歌词界面。
- 新增“播放完停止”播放模式。
- 新增 amll_ttml_tool 跳转入口，方便歌词编辑。

### Changed
- 项目名称完成迁移：TrueMusic → **kmgccc_player**；默认库路径调整为 `~/Music/kmgccc_player Library`（见 Notes）。
- 交互细节优化：增加图标动画。
- 优化 AMLL 歌词动画的应用细节，加入时间提前量，使歌词显示更及时，接近 apple 歌词效果。
- 深色/浅色模式下，歌词字重现可以分别设置，增强可读性。
- 歌词背景提供 磨砂玻璃/液态玻璃 切换选项。
- 完善资料库功能，增加基于歌手、专辑筛选歌曲功能，优化随机播放效果。

### Fixed
- UI 细节修复。
- 优化播放性能，减少占用。
- 歌词搜索的`lrc_to_ttml`转换脚本因为有去除平台声明、广告等前缀信息等正则匹配去除功能，在遇到特殊的歌词时，会造成歌词被误删。现已优化歌词转换脚本，减少歌词转换过程中的丢失或失败，但效果有限，所以现提供关闭剔除前缀信息的功能，歌词转换失败时可以尝试开启。
